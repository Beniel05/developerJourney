An HTML entity is a piece of text (a code) that displays reserved characters or invisible characters in a web browser.

Format SyntaxHTML entities always start with an ampersand (&) and end with a semicolon (;).
They can be written in two ways:
Entity Name: Easy to remember (e.g., & copy; for ©).
Entity Number: Browser-friendly code (e.g., & #169; for ©).
