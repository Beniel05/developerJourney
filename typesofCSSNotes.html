Types of CSS

CSS (Cascading Style Sheets) can be applied to HTML in three primary ways: Inline, Internal, and External. Each method varies in scope, reusability, and performance.

1. Inline CSS

Inline CSS applies styles directly to a specific HTML element using the style attribute.

<h1 style="color: blue;">Hello World</h1>
  • Scope: Only affects one element
  • Priority: Highest
  • Best For: Quick fixes
  • Drawback: Hard to maintain

2. Internal CSS

Defined inside a <style> tag in the <head> section.

<head>
  <style>
    p { color: red; }
  </style>
</head>
  • Scope: Entire page
  • Priority: Medium
  • Best For: Single-page projects
  • Drawback: Increases file size

3. External CSS

Uses a separate .css file linked to HTML.

<link rel="stylesheet" href="styles.css">
  • Scope: Multiple pages
  • Priority: Lowest
  • Best For: Large projects
  • Drawback: Extra request (but cached)

Summary Comparison

Feature Inline Internal External
Location Inside tag <style> in head Separate file
Priority Highest High Lowest
Reusability None Single page Multiple pages
Maintainability Difficult Moderate Easy
